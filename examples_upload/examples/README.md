# Example input and output files

These files contain fictional data only. They show the expected CSV layout;
they are not a complete runnable example because no video files are included.

- `sample_weather.csv`: weather CSV compatible with the default settings in the notebook.
- `sample_sunrise_sunset_04.csv`: one month of sunrise/sunset input in the expected format.
- `expected_output.csv`: an example of the CSV written after video timestamps are matched to selected weather records.
